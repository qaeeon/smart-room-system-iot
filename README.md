Project Description:
A smart room system where users can use a mobile app to control devices found in a room ie, lights, fans and curtains.

Microcontroller used:
- ESP32

Actuators used:
1. Lights
2. Fans
3. Curtains

Sensors used:
1. Light (LDR) sensor
2. Temperature & humidity sensor
3. Raindrop sensor

Dashboard used:
- Blynk IoT
